# Navigating Miramar dashboards

(13 July 2026).
Who is this article for?
Ideagen EHSQ (Formerly the DevonWay platform) Users
Miramar UI, record access, roles, and reporting authorities access restrictions apply.
The main landing page is the homepage (depicted below in the 'main region').
Dashboards consist of multiple components (regions) and can be configured to display information most relevant to each user.
Users can create and manage multiple dashboards. Each dashboard can have its own layout, and the content of each component can be customized. In addition to components, users can choose to display reports in one or more regions.
Users can select from a list of standard components, or select custom components that have been created by an organization's Admin.
Quickly toggle between dashboards using a keyboard shortcut
While viewing any dashboard, you can easily toggle through all dashboards by holding down 'Ctrl' + 'Shift' and pressing the up or down keys.
## Dashboard Components
The following are default components.
-
Assignment Summary: Graphical overview of assignments in a summary format grouped by Total, Unread, and Overdue. Toggle to also see assignments for teams and/or alternates.
-
Assignment List: View, filter, and drill down to assignments from a list or table display.
-
My Top Reports: Show the top five pinned reports or allow the system to automatically show the five most frequently used reports. Get quick insights on what items have changed in each report. Quickly drill down to lists of changed, overdue or unread items. Note: the counts displayed in My Top Reports may not be accurate for reports using the Other visualization option and right-side filters. Also, custom SSRS reports are not supported in My Top Reports.
-
Recent/Frequent: Show the fifty most recently viewed items on the first tab and the fifty most frequently viewed items on the second tab.
-
My Submissions/Recent: Show the last fifty items created on the first tab and the fifty most recently viewed items on the second tab
-
My Reports: See all reports organized in folders. Filter, find, and open reports with a few clicks. See all reports upon login without having to go to the Report Management page.
-
Recently Created Items: See what types of data users are putting into the system over time.
-
Up Next: Show the next item coming due in the assignments list.
Sharing components and dashboards
Administrators can create and share custom components and dashboards. These can be tailored to various roles, teams, or individuals, or they can be used company wide.
## Dashboard Customization
Each user can modify dashboards to meet their individual work needs. Each user can...
-
Add or remove components, select new components for display, change the dashboard layout, and add their own reports to their dashboard.
- Create as many new dashboards as they would like, which allows them to have unlimited access to organize as many useful work dashboards as they need.
- Choose which dashboard will be their home dashboard.
- Create browser bookmarks for each dashboard, allowing users easy access to often used dashboards.
-
Pin 5 dashboards directly to the left navigation bar.
Controlling personal dashboards
Dashboards cannot be locked down by administrators. Users can edit their own dashboards without affecting other dashboards, including shared dashboards.
## Create a new Dashboard
New dashboards can be created from scratch or by copying an existing dashboard.
To create a new Dashboard, click the dashboard button on the left navigation bar.
Add a new dashboard by clicking New Dashboard.
Duplicate an existing dashboard by clicking the 3-dot menu of the dashboard that you want to duplicate and select 'Duplicate'.
To change the layout of a dashboard simply click Select New Layout in the lower right corner.
Select the desired layout.
Once the layout has been selected, drag and drop Reports or Components into the desired region. To remove a selected Report or Component, simply drag a new element into the box or click the X in the center of the box to cancel the selection. Be sure to click 'Save' after making your selections.
Don't forget to Save the changes.
## Editing Dashboards
To edit the dashboard, click the dashboard button on the side navigation bar. Hover the cursor over the dashboard and click the pencil (edit) icon.
Select a new layout, if desired.
Once the new dashboard has been saved users can choose to pin it to the left navigation bar (with a custom icon), choose to have to be their home dashboard (open at login), rename it, duplicate it, or delete it.
Don't forget to Save the changes.
## Pinning Dashboards
For easy access, Pin a dashboard and assign it a new icon. Pinned dashboards will be displayed as the selected icon in the left navigation bar. Users can pin up to five dashboards.
